# FleetMate — Product Context Index

This folder contains the **stable product knowledge** that grounds every initiative.
All Cursor commands and agents reference these files before generating any artifact.

| File | Purpose |
|------|---------|
| `01-one-pager.md` | What the product is, who it serves, why it matters |

## Usage rule

Before creating any initiative artifact, read:
1. `01-one-pager.md` — understand the product
2. `04-personas-and-segments.md` — know who is affected
3. `02-strategy-and-okrs.md` — check strategic alignment
4. `03-metrics-and-definitions.md` — use correct metric names
